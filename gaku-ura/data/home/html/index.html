php

gaku-ura/conf ディレクトリには共通部分のソースコードがありますが、便利なライブラリでもあります。

index.phpのようにgaku-ura/main ディレクトリのphpを呼び出し、gaku-ura/main に本体のphpを作ってgaku-ura/conf内のライブラリを使うと簡単に共通html/cssテンプレートを使う準備が整います。


css/javascript

疑似命令で他のファイルをincludeしたりできます。


html/md

GakuUra::htmlf()を使って出力するhtmlには、疑似命令で読み込むcss/javascriptや、タイトル・説明・検索インデックスなどを指定出来ます。

また、html/mdファイル中の「{KEY}」が、'KEY'がキーの連想配列をGakuUra::htmlf()で指定すると、対応する値に置換される機能もあります。

これらの設定は、index.php(トップページ)で使うファイルには既に構成済みです。


詳細はgaku-ura配布ページからご確認を。
